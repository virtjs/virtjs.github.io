Emulators. Emulators everywhere. Start now.

Virtjs allows you to create embeddable emulators on a variety of architectures. More than that, it exposes conventioned public API so you can easily plug them into your own projects.

Open Source

Virtjs is released under the MIT license, giving it free of charge to every open-source project that you could think of. The world is yours!

Modular

Virtjs aims to give you full leverage on the running emulator. It allows you to write your own devices, should you need it, and to directly wire them into the core engines.

Modern

Virtjs has been made with the full power of ES6, the next iteration of the Javascript programming language. It gives us a pretty simple source code. We're one of the very first libraries to jump aboard!

// By the way, this exact code is used for launching the emulator above :)import { KeyboardInput }       from 'virtjs/devices/inputs/KeyboardInput';import { WebGLScreen }         from 'virtjs/devices/screens/WebGLScreen';import { AnimationFrameTimer } from 'virtjs/devices/timers/AnimationFrameTimer';import { fetch }               from 'virtjs/tools';import { Engine }              from 'virtjs-gb/Engine';import { inputs }              from 'virtjs-gb/constants';var input = new KeyboardInput( { inputs : inputs } );var screen = new WebGLScreen( { canvas : document.querySelector( '.example .screen' ) } );var timer = new AnimationFrameTimer( );var engine = window.engine = new Engine( { devices : {    input : input,    screen : screen,    timer : timer} } );fetch( '/roms/tuff.gb' ).then( rom => {    engine.loadArrayBuffer( rom );} );
